# **App Name**: AsistenciaPro CRM

## Core Features:

- Gestión de Empresas y Cuentas Clientes: Permite registrar y gestionar las diferentes empresas de asistencia y sus cuentas de cliente asociadas (e.g., IKE Asistencia -> Coomeva).
- Registro y Seguimiento de Solicitudes de Servicio: Crea nuevas solicitudes para servicios (plomería, electricidad, cerrajería, etc.) y actualiza su estado (pendiente, asignado, en progreso, completado).
- Asignación Simple de Técnicos: Funcionalidad básica para asignar solicitudes de servicio a técnicos disponibles o específicos.
- Panel de Control de Servicios Activos: Una vista consolidada para monitorear el estado actual de todas las solicitudes de servicio activas y pendientes.
- Historial y Notas Detalladas de Servicio: Mantiene un registro de todos los servicios completados para cada cuenta cliente y permite agregar notas detalladas por cada intervención.
- Generador de Resúmenes de Notas de Servicio: Una herramienta de IA que automáticamente genera resúmenes concisos a partir de notas de servicio detalladas para una revisión rápida y eficiente.

## Style Guidelines:

- Primary color: Professional Blue (#1F5BCC) for a sense of trust and efficiency.
- Background color: Very light desaturated blue (#ECF1FC) to maintain a clean and airy feel, enhancing readability.
- Accent color: Cyan-Teal (#4DBBD1) to provide visual cues and highlight interactive elements without being overpowering, offering a subtle contrast.
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral, and highly readable qualities, suitable for business applications with varying text lengths.
- Use clear, professional, and easily recognizable vector icons related to service categories and CRM functions. Favor a consistent line-based style for modernity.
- Clean, organized, and responsive dashboard-style layouts with well-defined sections for service requests, client management, and performance overview, optimized for desktop and tablet usage.
- Subtle, non-distracting animations for transitions between views and feedback on user actions (e.g., successful submission, status updates) to enhance user experience.
